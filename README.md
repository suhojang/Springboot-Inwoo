### Spring-boot 를 활용한 web server

#### Intro
> gradle, Thymeleaf, mail, lombok, mybatis, spring security, logback-spring을 활용하여 개발
> 
> korean, english 지원
> 
> 고객용 웹 사이트와 관리자용 웹 사이트 제공
